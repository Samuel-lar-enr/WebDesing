Instrucciones de Prueba

Para verificar que funciona correctamente:

  1. Campos vacíos: Los campos requeridos deben tener un borde diferente
  2. Email inválido: Escribe "texto" → debe mostrar estilo de error
  3. Email válido: Escribe "usuario@ejemplo.com" → debe mostrar estilo válido
  4. Teléfono: Solo debe aceptar 9 números
  5. Contraseña: Debe requerir mínimo 8 caracteres
  6. Focus: Al hacer clic en un campo, su estilo debe cambiar
  7. Campos opcionales: Deben verse visualmente diferentes a los requeridos

Pseudo-clases que debes usar en tu CSS

  • :focus - Cuando el campo está seleccionado
  • :valid - Cuando el valor es válido (borde verde)
  • :invalid - Cuando el valor es inválido (borde rojo)
  • :required - Para campos obligatorios (borde izquierdo naranja)
  • :optional - Para campos opcionales (borde izquierdo gris)
  • :hover - Para el botón cuando pasas el ratón
